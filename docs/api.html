Quad-Core API Reference

Backend server runs at http://localhost:8000 (dev) or configured via VITE_PYTHON_API.

Python Backend Endpoints

GET /health

Health check endpoint. Returns server status and version info.

{
  "status": "ok",
  "version": "1.0.0",
  "uptime": 1719232800.0,
  "timestamp": "2026-06-24T14:30:00Z"
}

POST /api/generate-level

Generate encrypted level payload. The server creates a procedural level, encrypts it, and returns the ciphertext. The client never sees the raw seed or level layout.

// Request
{
  "level_index": 5,
  "tier": 2,
  "client_fingerprint": "fp_abc123_def456"
}

// Response
{
  "encrypted_payload": "gAAAAAB...encrypted...",
  "integrity_hash": "a1b2c3d4e5f6...",
  "seed_hash": "f9e8d7c6..."
}
FieldTypeDescription
encrypted_payloadstringFernet-encrypted level data (decrypted by Rust WASM)
integrity_hashstringSHA-256 of encrypted payload + server salt
seed_hashstringDeterministic seed identifier

GET /api/auth/wasm-secrets

Provide obfuscated physics constants to the WASM module. These are not the actual constants — the WASM module derives gravity/friction from them at runtime via chaotic functions.

{
  "gravity_hash": 11.772,
  "friction_seed": 4815162342,
  "validation_token": 3.1415926535
}
FieldTypeDescription
gravity_hashfloatScaled gravity value (actual gravity derived chaotically)
friction_seedintSeed for friction coefficient derivation
validation_tokenfloatSession token for frame validation

POST /api/auth/validate-frame

Validate a physics frame hash. The Rust WASM module generates a chaotic validation hash for each frame. The server verifies it matches expectations, enabling anti-cheat without full physics re-simulation (Federated Physics Validation patent method).

// Request
{
  "frame_hash": 0.8472,
  "level_index": 5,
  "client_fingerprint": "fp_abc123",
  "expected_gravity_seed": 3.14159
}

// Response
{
  "valid": true,
  "server_time": 1719232800.0
}

Rate Limiting

Level generation is rate-limited to 30 requests per 60-second window per client fingerprint. Exceeding this returns HTTP 429.
